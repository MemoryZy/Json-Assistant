This operation detects that the string values themselves are valid JSON and expands those string values into JSON objects/arrays.

此操作会检测到字符串值本身是有效 JSON 的情况，并将这些字符串值展开为 JSON 对象/数组。

Example:


{
  "content": "{\"hobbies\":[\"reading\"]}"
}

After the quick-fix is applied:


{
  "content": {
    "hobbies": [
      "reading"
    ]
  }
}
