# Json Assistant

[English](./README.md) / 简体中文

基于 IntelliJ IDE 的 JSON 工具插件。

[![IntelliJ IDEA][shields:idea]][jb:link]

<br/>

## 功能 🎉
- JSON 展示窗口
- JSON 格式化
- JSON 压缩
- JSON 结构化
